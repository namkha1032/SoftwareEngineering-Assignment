gõ vào terminal để tải:

pip install -r requirements.txt

Các bước để chạy:
gõ vào terminal

python manage.py migrate

python manage.py runserver

nếu bị redirect tới trang login
Username: admin
Password: 12345

python manage.py createsuperuser
rồi tạo username password mới
